## Free Courses

### **freeCodeCamp - Database Design Course**
- **Platform**: YouTube / freeCodeCamp.org
- **Duration**: ~4 hours
- **Level**: Beginner to Intermediate
- **Topics Covered**:
  - Entity-relationship diagrams
  - Normalization (1NF through 3NF)
  - SQL fundamentals
  - Practical database design
- **Link**: [freeCodeCamp YouTube](https://www.youtube.com/c/Freecodecamp)
- **Why Take It**: Comprehensive free introduction with hands-on examples

### **Khan Academy - Intro to SQL**
- **Platform**: Khan Academy
- **Duration**: Self-paced (~10 hours)
- **Level**: Beginner
- **Topics Covered**:
  - SQL basics (SELECT, WHERE, JOIN)
  - Data aggregation
  - Database design basics
  - Interactive exercises
- **Link**: [khanacademy.org/computing/computer-programming/sql](https://www.khanacademy.org/computing/computer-programming/sql)

### **Codecademy - Learn SQL (Free Version)**
- **Platform**: Codecademy

- **Level**: Beginner
- **Topics Covered**:
  - SQL syntax and queries
  - Aggregate functions
  - Multiple tables and joins
  - Hands-on projects
- **Link**: [codecademy.com/learn/learn-sql](https://www.codecademy.com/learn/learn-sql)

### **Stanford Online - Databases: Relational Databases and SQL**

## Tips for Choosing a Course

1. **Match Your Learning Style**:
   - Video learner → YouTube tutorials, Udemy, Coursera
   - Hands-on learner → SQLZoo, HackerRank, DataCamp
   - Academic learner → University courses (MIT, Stanford)

2. **Consider Your Budget**:
   - Free options: Khan Academy, freeCodeCamp, SQLZoo, MIT OCW
   - Affordable: Udemy ($10-$20 on sale)
   - Subscription: Coursera, Pluralsight, DataCamp

3. **Check for Certificates**:
   - LinkedIn Learning, Coursera, DataCamp offer completion certificates
   - Useful for resume building

4. **Read Reviews**:
   - Always check course ratings and recent reviews
   - Look for updated content (SQL standards evolve)

5. **Trial Periods**:
   - Many platforms offer 7-day free trials
   - Use trials to test multiple platforms
