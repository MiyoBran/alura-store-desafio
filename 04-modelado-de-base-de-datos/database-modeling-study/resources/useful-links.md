## Tools and Software

### **Database Design and Modeling Tools**

- **dbdiagram.io**: [dbdiagram.io](https://dbdiagram.io/)
  - Quick and simple database design tool using DBML
- **draw.io (diagrams.net)**: [draw.io](https://www.draw.io/)
  - Free diagramming tool with ER diagram templates
- **Lucidchart**: [lucidchart.com](https://www.lucidchart.com/)
  - Professional diagramming with database templates
- **MySQL Workbench**: [mysql.com/products/workbench](https://www.mysql.com/products/workbench/)
  - Official MySQL design tool
- **pgModeler**: [pgmodeler.io](https://pgmodeler.io/)
  - PostgreSQL database modeler
- **DBeaver**: [dbeaver.io](https://dbeaver.io/)
  - Universal database tool with ER diagram support
- **Vertabelo**: [vertabelo.com](https://www.vertabelo.com/)
  - Online database design tool
- **QuickDBD**: [quickdatabasediagrams.com](https://www.quickdatabasediagrams.com/)
  - Fast database diagram tool

### **Text-Based Modeling**

- **PlantUML**: [plantuml.com](https://plantuml.com/)
  - Create diagrams from plain text
- **Mermaid**: [mermaid.js.org](https://mermaid.js.org/)
  - Markdown-inspired text-based diagramming
- **DBML**: [dbml.dbdiagram.io](https://www.dbml.dbdiagram.io/)
  - Database markup language

### **Database Clients and GUIs**

- **TablePlus**: [tableplus.com](https://tableplus.com/)
  - Modern database GUI for multiple databases
- **Beekeeper Studio**: [beekeeperstudio.io](https://www.beekeeperstudio.io/)
  - Open-source SQL editor and database manager
- **DataGrip**: [jetbrains.com/datagrip](https://www.jetbrains.com/datagrip/)
  - JetBrains database IDE
- **Sequel Pro** (macOS): [sequelpro.com](https://www.sequelpro.com/)
  - MySQL database management
- **HeidiSQL** (Windows): [heidisql.com](https://www.heidisql.com/)
  - Lightweight MySQL/PostgreSQL client

### **Migration and Schema Management**

- **Flyway**: [flywaydb.org](https://flywaydb.org/)
  - Database migration tool
- **Liquibase**: [liquibase.org](https://www.liquibase.org/)
  - Database schema change management
- **Alembic**: [alembic.sqlalchemy.org](https://alembic.sqlalchemy.org/)
  - Database migration for SQLAlchemy (Python)
