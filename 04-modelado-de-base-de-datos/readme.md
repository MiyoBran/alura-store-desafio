# Curso: Modelado de bases de datos - entidades, relaciones y atributos

Este es el proximo curso dentro de la ruta de Base de Datos.

## Objetivo
Comprender el modelado conceptual y logico de datos para construir modelos claros y
correctos que sirvan de base para el diseno de bases de datos.

## Lo que voy a aprender
- Identificar el modelo conceptual de un banco de datos.
- Diferenciar un modelo de alto nivel de uno de bajo nivel.
- Aplicar tecnicas de modelado de datos.
- Distinguir el Modelo Entidad-Relacion (MER) del Diagrama Entidad-Relacion (DER).
- Elaborar un modelo en diagrams.net.

## Entregables esperados
- Un modelo MER documentado.
- Un DER dibujado en diagrams.net.
- Una breve explicacion de decisiones de modelado.

## Requisitos
- Conocimientos basicos de bases de datos.
- Acceso a diagrams.net (web o desktop).

## Notas
- Mantener consistencia en nombres de entidades y atributos.
- Documentar supuestos y reglas de negocio clave.
