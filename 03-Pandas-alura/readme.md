# Curso: Pandas - conociendo la biblioteca

Este es el proximo curso que realizare dentro de la ruta de Data Science.

## Objetivo
Aplicar los recursos principales de Pandas para analizar datos y construir un proyecto
con resultados claros y reproducibles.

## Lo que voy a aprender
- Cargar y explorar datos con Pandas.
- Seleccionar y filtrar informacion relevante.
- Tratar datos nulos y valores inconsistentes.
- Remover filas y columnas no necesarias.
- Crear y transformar columnas.
- Generar graficas basicas para el analisis exploratorio.

## Entregables esperados
- Un notebook con el flujo completo de analisis.
- Visualizaciones que expliquen los hallazgos.
- Un breve resumen con conclusiones y siguientes pasos.

## Requisitos
- Python 3.x.
- Jupyter Notebook o VS Code con soporte de notebooks.
- Librerias: pandas, matplotlib (y opcionalmente seaborn).

## Datos
Si el curso usa datos del repositorio, la carpeta sugerida es:

- Documentar supuestos y decisiones durante la limpieza de datos.
