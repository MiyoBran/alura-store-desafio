# Python para Data Science: primeros pasos

Este paquete contiene el material del curso introductorio de Python aplicado a Data Science. Incluye cuadernos con teoria, ejemplos guiados y ejercicios practicos para consolidar los fundamentos del lenguaje.

## Contenido del paquete
- `Curso_1_Python_Data_Science.ipynb`: cuaderno principal con explicaciones y ejemplos paso a paso.
- `Ejercicios_Python_Primeros_Pasos.ipynb`: ejercicios para practicar lo visto en el curso.

## Lo que aprenderas
- Comandos basicos del lenguaje Python.
- Tipos de variables y sus diferencias.
- Manipulacion de tipos basicos de datos.
- Estructuras condicionales.
- Estructuras de repeticion.
- Estructuras de datos y su uso.
- Relacion entre estructuras del lenguaje y proyectos de datos.

## Como usarlo
1. Abre los cuadernos en Jupyter o VS Code.
2. Ejecuta las celdas en orden y toma notas.
3. Resuelve los ejercicios y compara tus resultados.
