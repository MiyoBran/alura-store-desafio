# Python para Data Science: bibliotecas, funciones y excepciones

Este paquete contiene el material del curso enfocado en el uso de bibliotecas, funciones y el manejo de errores en Python para Data Science. Encontraras cuadernos con teoria, ejemplos y ejercicios practicos.

## Contenido del paquete
- `Bibliotecas_y_funciones_V2_0.ipynb`: cuaderno principal con conceptos y ejemplos.
- `Ejercicios_Estructuras_Anidadas.ipynb`: practica de estructuras compuestas.
- `Ejercicios_Trabajando_con_Excepciones.ipynb`: manejo de errores y excepciones.
- `Ejercicios_trabajar_con_funciones,_estructuras_de_datos_y_excepcione_.ipynb`: repaso integrador.

## Lo que aprenderas
- Funciones de bibliotecas y paquetes en Python.
- Funciones integradas (built-in) y sus usos.
- Creacion de funciones personalizadas.
- Estructuras de datos compuestas y anidadas.
- Comprensiones de listas y diccionarios.
- Tipos de errores y excepciones.
- Estrategias para evitar comportamientos indeseados.

## Como usarlo
1. Abre los cuadernos en Jupyter o VS Code.
2. Ejecuta las celdas en orden y revisa los ejemplos.
3. Resuelve los ejercicios y verifica tus resultados.
